# 🚀 LevelUp Studio — Game Collection

Welcome to **LevelUp Studio**, a responsive and modern web game collection built for the **2026 School Expo**. This project consolidates five classic and engaging arcade/adventure games into a unified, visually stunning landing page experience.

The entire project is built using vanilla **HTML5, CSS3, and JavaScript**—no external frameworks, build steps, or installations are required!

---

## 🎮 The Games

1. **🧱 Tetris (`tetris.html`)**
   - The timeless block-stacking puzzle game. Rotate and drop shapes, clear lines, and try to survive as the speed increases!
2. **📝 Hangman (`hangman.html`)**
   - A word-guessing brain teaser. Guess letters to reveal the secret word before the stick figure is fully drawn.
3. **🐍 Snake (`snake.html`)**
   - A retro arcade favorite. Guide the snake to eat red apples, grow longer, and avoid crashing into walls or your own tail.
4. **🃏 Memory Cards (`memory-cards.html`)**
   - A concentration-based card matching game featuring **10 levels** of increasing grid sizes and difficulty.
5. **🔥 Realm of Embers (`fantasy-adventure.html`)**
   - A rich, text-based fantasy adventure RPG. Equip your hero, manage stats (HP/MP), navigate choices, and conquer the quest.

---

## ✨ Key Features

- **📺 Full-Screen Hero Banner**: A custom-designed, responsive space banner fitting the viewport height perfectly.
- **📱 Single-Row Grid Layout**: The game collection cards are displayed in a clean, side-by-side row on desktop and a smooth, snap-scrolling touch-swipe carousel on mobile/tablet viewports.
- **☀️ / 🌙 Cohesive Theme Engine**:
  - Switch between light and dark mode on any page using the glassmorphism toggle button in the top-right corner.
  - **Shared State**: The theme preference is saved in the browser's `localStorage`. Switching the theme on the landing page automatically updates all the games, and vice versa!
  - **Custom Aesthetics**: Games feature custom light/dark styling (including canvas grid colors, parchment sepia fantasy modes, and custom gallows colors).
- **🔙 Easy Navigation**: Every game has a dedicated, fixed "Back to Home" button styled to match the individual game's theme.
- **⚡ Super Lightweight**: Smooth micro-animations, glassmorphic UI, responsive layouts, and animations powered entirely by vanilla code.

## 🚀 How to Run Locally

Since this collection is built entirely on standard frontend technologies, there's no server or package installation required!

1. **Download/Clone** this repository directory.
